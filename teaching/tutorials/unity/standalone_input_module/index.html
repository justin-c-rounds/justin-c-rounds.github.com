Unity 2d: Standalone Input Module

  1. Create a new 2d project.
  2. Add the Standalone Input Module component to the Main Camera: Add Component → Event → Standalone Input Module.
  3. Add the Physics 2d Raycaster component to the Main Camera: Add Component → Event → Physics 2d Raycaster.
  4. Create a new sprite: GameObject → 2D Object → Sprite.
  5. Add the Box Collider 2D component to the sprite: Add Component → Physics 2D → Box Collider 2D.
  6. Add a custom script to the Sprite with a public function to handle a mouse pointer event, e.g.:
     public void Drag (BaseEventData data) {
   PointerEventData p = (PointerEventData)data;
   Vector3 vp = Camera.main.WorldToScreenPoint (transform.position);
   Vector3 d = new Vector3 (p.delta.x, p.delta.y, 0);
   transform.position = Camera.main.ScreenToWorldPoint (vp + d);
 }
  7. Add an Event Trigger component to the Sprite: Add Component → Event → Event Trigger.
  8. Click the "Add New Event Type" button in the Event Trigger component and select the appropriate event (e.g. Drag).
  9. Click the little target button in the event type box and select the Sprite asset.
  10. Click the dropdown which says "No Function" and select the public function you created.
  11. Test the mouse input on your sprite.
