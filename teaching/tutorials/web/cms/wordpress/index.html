WordPress

WordPress is a popular content management system used for blogs, webcomics, online stores, and simple websites. It supports themes for controlling the display of content, widgets for providing different methods of accessing content, users and roles for controlling administration and authoring of content, and plugins to extend functionality.

WordPress is a web application written in PHP, running on a web server with a MySQL database.

Themes

WordPress themes are templates which define the way content is placed into HTML and CSS rules defining how the content should be displayed. Themes can be customized within the WordPress admin interface.

Widgets

Widgets are areas within the site where links to content or content metadata are displayed. Widgets can be configured within the WordPress admin interface.

Users and Roles

Multiple users can be granted access to various aspects of a WordPress site. Roles define which users have access to which aspect of the site.

Plugins

WordPress plugins extend the core functionality of WordPress and are installable and configurable within the admin interface.
