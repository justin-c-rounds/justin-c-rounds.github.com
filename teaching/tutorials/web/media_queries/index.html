Web Design & Development: Media Queries

Introduction

Media queries allow web pages to apply different CSS rules based on device orientation, screen size, and other criteria.

Usage

There are two methods for using media queries:

  1. As part of a <link> tag.

     <link rel="stylesheet" media="(max-width: 800px)" href="example.css" />

    This indicates to the browser that the CSS contained in example.css should only be applied when the browser window is less than or equal to 800 pixels wide.

  2. Within a stylesheet.

     @media (min-width: 600px) {
   .facet_sidebar {
     display: none;
   }
 }

    This indicates to the browser that the CSS contained within the @media block should only be applied when the browser window is greater than or equal to 600 pixels wide.

While both of these methods work, using the media attribute of the <link> tag is less optimal because the browser will have to download the additional stylesheet. In general, it is best to reduce the number of files required by the browser to render a web page, in addition to keeping file sizes as small as possible.
