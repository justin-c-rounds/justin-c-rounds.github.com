Web Design & Development: Responsive Design

Responsive Design is the practice of designing websites to respond appropriately to varying devices, screen sizes, and user preferences.

Fluid layout

A fluid (aka liquid) layout allows content to flow elegantly regardless of browser window size.

See an example of a fluid layout.

Implementation

There are several ways to create fluid layouts, but in general, follow these guidelines:

  1. Never use px units in CSS, only use em, rem, or percentage-based sizes.
  2. Learn how CSS positioning works. Absolute positioning has a bad reputation, but it can be used to easily create layouts which stretch with changes in browser window size.
  3. Allow content to flow like water. To change its shape, adjust its container.

See an example of a fluid layout.

Media Queries

Media queries are used to apply different CSS rules to web pages based on display device, screen size, orientation, and other criteria.

Flexible Images

Flexible images do not break the layout due to fixed pixel sizes and will shrink (or stretch) to accommodate their containing elements.
